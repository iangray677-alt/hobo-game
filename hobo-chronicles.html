Phone Saving / Viewing Instructions
  1. Keep this HTML file and your exported hobo-chronicles-save.json in the same cloud folder, such as iCloud Drive, Google Drive, OneDrive, or your phone's Files app.
  2. Open the HTML file in your phone browser. If your latest data is not showing, use Import Save JSON and choose the save file.
  3. After editing on your phone, tap Export Save JSON. Save or move the downloaded JSON back into your planner folder.
  4. On another device, open the HTML file and import the newest JSON save. The HTML is the app; the JSON is your portable data.
If the planner nodes do not appear, this file is being previewed instead of opened in a browser. On iPhone, open the HTML in Safari/Chrome from Files or a cloud drive.
